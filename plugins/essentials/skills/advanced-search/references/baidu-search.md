# Baidu Web Search — 参数参考

百度 Web 搜索是百度千帆 (Qianfan) 平台提供的 Web 搜索 MCP 服务，通过 HTTP 传输，使用 `${BAIDU_API_KEY}` 认证。

**每日免费额度：50 次。** 适合国内非技术信息的搜索，覆盖百度百家号、百度百科、CSDN、知乎等中文来源。

## MCP 工具

`mcp__baidu_web_search__webSearch`
→ 实际调用时使用工具名 `baidu_web_search` 调用 `webSearch`

## 参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `query` | string | (必填) | 搜索查询字符串。支持自然语言关键词 |
| `count` | integer | 10 | 返回结果数量，范围 1-50 |
| `freshness` | string | (可选) | 时间范围过滤。支持两种格式：①相对时间 `pd`(过去24小时)、`pw`(过去7天)、`pm`(过去31天)、`py`(过去365天)；②具体日期范围 `YYYY-MM-DDtoYYYY-MM-DD`（如 `2026-01-01to2026-06-08`） |

**无 `summary`、`search_depth`、`include_domains` 等高级参数。** 这是一个精简的基础搜索接口。

## 响应结构

```json
{
  "results": [
    {
      "Title": "页面标题",
      "URL": "完整 URL",
      "Content": "页面内容摘要/片段",
      "site_name": "来源站点名称（隐含在 URL 的主机名中）",
      "published_date": "发布日期（部分结果包含）"
    }
  ]
}
```

- `Title` — 页面标题，可用于结果展示
- `URL` — 完整链接，可直接点击访问
- `Content` — 页面摘要，通常是 2-3 句关键内容
- 部分结果包含隐性发布日期信息（体现在内容片段中）

## 特点与优势

| 维度 | 评价 |
|------|------|
| **中文内容覆盖** | ★★★★★ — 百度百家号、百度百科、CSDN、知乎、搜狐等中文来源覆盖极好 |
| **国际内容** | ★★☆☆☆ — 可返回 GitHub 等国际站点，但覆盖面有限 |
| **速度** | ★★★★★ — 百度搜索基础设施，响应快 |
| **结果丰富度** | ★★★☆☆ — 返回标题、URL、摘要，无完整内容提取、无相关性评分 |
| **高级过滤** | ★★☆☆☆ — 支持 freshness 时间过滤（相对时间+日期范围），但不支持域名过滤、搜索深度控制 |
| **成本** | 🆓 — 每日 50 次免费额度 |

**定位：** 作为五引擎搜索中的中文广覆盖补充层，强化国内非技术类信息的搜索。与 Bailian（百炼，聚焦阿里生态内容）形成互补 — 百度覆盖百度系内容生态（百家号、百度百科、百度贴吧等），Bailian 覆盖阿里系/中立方内容。Bocha 则提供语义理解和精确日期过滤。

## 与其他引擎的互补关系

Baidu 与 Bailian 都擅长中文搜索，但来源侧重不同：

| 来源类型 | Baidu | Bailian |
|----------|-------|---------|
| 百度百家号 | ✅ 强 | ⚠️ 部分 |
| 百度百科 | ✅ 强 | ⚠️ 部分 |
| CSDN | ✅ | ✅ |
| 知乎 | ✅ | ✅ |
| 腾讯云/阿里云 | ⚠️ 部分 | ✅ 强 |
| 搜狐/网易/新浪 | ✅ 强 | ⚠️ 部分 |

两者同时搜索可以最大化中文来源覆盖。

## 搜索技巧

- **中文查询优先** — 百度对中文查询的返回质量最高
- **利用 freshness 参数** — 这是 Bailian 没有的能力，时效性查询时使用 `pw` (过去7天) 或 `pd` (过去24小时)
- **国内非技术话题** — 生活、娱乐、政策、社会新闻等百度搜索效果好；技术类话题建议同时依赖 Tavily
- **count 控制** — 默认 10 条，上限 50。日常搜索保持 5-10 条以节省额度
- **额度管理** — 每日仅 50 次免费调用，避免在单次搜索中过度消耗（如 count=50）。常规搜索使用 `count=5`，深度搜索使用 `count=10`

## 局限

- **每日 50 次免费额度** — 超出后需要付费或等待次日重置。合理安排搜索次数
- **无内容提取能力** — 无法像 Tavily 那样获取页面正文
- **无域名过滤** — 不支持 include_domains/exclude_domains
- **无搜索深度控制** — 不像 Tavily 有 basic/advanced 深度选择
- **无相关性评分** — 不像 Tavily 返回 relevance score
- **国际内容覆盖面有限** — 英文查询建议更多依赖 Tavily 和 WebSearch
- **无显式发布日期字段** — 部分结果的日期信息隐含在片段文本中，不如 Bocha 的 `published_date` 精确
- **API Key 依赖** — 需要用户配置 `BAIDU_API_KEY`，未配置时该引擎不可用
