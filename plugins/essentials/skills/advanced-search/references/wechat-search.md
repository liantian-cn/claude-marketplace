# 搜狗微信搜索 — 参数参考

通过搜狗微信搜索 (weixin.sogou.com) 获取微信公众号文章。**无需 API Key**，使用网页抓取方式获取搜索结果。

微信公众号内容生态包含大量高质量中文长文，覆盖技术分析、行业洞察、政策解读、学术讨论等领域，是其他搜索引擎难以触及的独特信息来源。

## 调用方式

WeChat 搜索没有 MCP 服务，通过 **直接 Bash 命令** 执行本地 Python 脚本：

```bash
python plugins/essentials/skills/advanced-search/scripts/search_wechat.py "关键词" -n 数量
```

**关键区别：** WeChat 与其他五个引擎不同 — 它不使用 MCP 工具，不需要子代理 (subagent)。在并行执行阶段直接通过 Bash 调用脚本，读取 stdout 的 JSON 输出。

## 参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `query` | string | (必填) | 搜索关键词。**仅支持中文查询** — 微信公众号内容几乎全为中文。如果用户使用英文查询，应翻译为中文关键词后再搜索 |
| `-n, --num` | integer | 10 | 返回结果数量，范围 1–50。每页约 10 条结果，脚本自动翻页 |
| `-o, --output` | string | (可选) | 输出 JSON 文件路径。不指定则仅输出到 stdout |

**无 `freshness`、`search_depth`、`site:` 等高级过滤参数。** 这是一个基础的网页抓取搜索工具。

## 响应结构

```json
{
  "query": "搜索关键词",
  "total": 10,
  "articles": [
    {
      "title": "文章标题",
      "url": "搜狗中间链接 (https://weixin.sogou.com/link?...)",
      "summary": "文章摘要/片段",
      "source": "公众号名称",
      "published_at": "2026-06-13 15:34:55"
    }
  ]
}
```

- `title` — 文章标题
- `url` — 搜狗中间链接（非微信原文 `mp.weixin.qq.com` 直链）。搜狗会对微信文章 URL 进行包装，点击后通过搜狗跳转至原文
- `summary` — 文章摘要，通常 2–3 句关键内容
- `source` — 微信公众号名称（如 "第一财经"、"新京报"）
- `published_at` — 发布时间，格式 `YYYY-MM-DD HH:MM:SS`，从页面时间戳解析

**错误响应：**
```json
{
  "query": "关键词",
  "total": 0,
  "articles": [],
  "error": "错误描述"
}
```

## 与其他引擎的互补关系

WeChat 搜索的核心价值在于**独占内容** — 微信公众号文章是封闭生态，不被任何通用搜索引擎索引：

| 内容类型 | WeChat | 其他引擎 |
|----------|--------|----------|
| 微信公众号长文 | ✅ **独有** | ❌ 完全不覆盖 |
| 百度百家号 | ❌ | ✅ Baidu 独有 |
| CSDN / 知乎 | ❌ | ✅ Baidu + Bailian |
| 阿里云 / 腾讯云技术文档 | ❌ | ✅ Bailian |
| 国际新闻 / 学术论文 | ❌ | ✅ Tavily + WebSearch |
| 政府公告 / 政策文件 | ⚠️ 部分 | ✅ Bailian + Baidu |

**最佳使用场景：**
- 中国本土深度行业分析、商业洞察
- 科技/互联网公司动态的中文深度解读
- 学术/专业领域的公众号长文
- 政策解读类内容（部分政府/智库通过公众号发布）
- 社会热点事件的多角度评论分析

## 搜索技巧

- **纯中文关键词** — 微信公众号内容几乎全为中文。英文查询需先翻译为中文关键词
- **搜索词精炼** — 2–4 个核心关键词效果最好，过长的自然语言查询会降低匹配度
- **翻页限制** — 最多 5 页（约 50 条），更多结果需要缩小搜索范围
- **Cookie 获取** — 脚本自动从 `v.sogou.com` 获取 Cookie，无需手动处理
- **结果 URL** — 返回的链接是搜狗中间页（`weixin.sogou.com/link?…`），不是 `mp.weixin.qq.com` 直链。如需原文直链需要二次解析（有反爬风险，不推荐）
- **与其他引擎互补** — WeChat 提供独特的中文长文视角，应与 Baidu（国内短内容）、Bailian（阿里生态技术内容）、Tavily（国际深度内容）同时使用

## 局限与风险

- **反爬虫机制** — 搜狗微信有严格的反爬虫策略，可能导致请求失败或返回空结果。如果失败，标注 `[WeChat: unavailable]` 并继续使用其他五个引擎
- **无时间过滤** — 无法按时间范围筛选，只能通过关键词中包含时间词（如 "2026"、"最新"）来间接影响结果
- **无全文提取** — 仅返回标题和摘要，不提供正文抓取能力。如需全文，用户需手动访问链接
- **仅中文内容** — 对英文查询无效，英文搜索应主要依赖 Tavily 和 WebSearch
- **翻页上限 50 条** — 5 页 × 10 条 = 50 条上限，超出部分无法获取
- **URL 非直链** — 返回搜狗中间链接而非微信原文链接，但不影响用户点击访问
- **请求速率限制** — 脚本内置 0.5–1.5 秒页间随机延迟，避免触发反爬。过于频繁的搜索可能导致 IP 被暂时封禁
- **不稳定** — 搜狗可能随时调整页面结构，导致解析失效。这是所有无 API 抓取工具的固有限制
