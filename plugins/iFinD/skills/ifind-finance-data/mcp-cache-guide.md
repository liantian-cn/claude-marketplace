# 金融数据 MCP 查询缓存约定

> **适用范围**: 所有使用 iFinD MCP 的 skill 均遵循此约定，以减少重复 MCP 请求。

## 缓存目录

按数据域分目录：`./金融数据缓存/[数据域]/`

```
./金融数据缓存/
├── 股票数据/
├── 基金数据/
├── 宏观行业/
├── 新闻公告/
├── 债券数据/
├── 港美股数据/
└── 指数板块/
```

## 缓存流程

执行任何 MCP 查询前，按以下步骤：

1. **检查缓存**: 检查 `./金融数据缓存/[域]/[缓存文件名]` 是否存在
2. **命中**: 若文件存在且内容有效（见下文刷新策略） → 直接读取，跳过 MCP 调用
3. **未命中**: 调用 MCP 工具（或脚本兜底）→ 将返回结果以 Markdown 格式保存到对应缓存文件 → 继续使用

## 缓存文件格式

每个缓存文件包含：

```markdown
# [数据项名称]

**查询时间**: YYYY-MM-DD HH:MM
**数据来源**: [MCP 工具完整名 / 脚本调用]
**查询内容**: [用户原始查询描述]

## 查询结果

[语义化的 MCP 返回内容]
```

## 缓存文件命名

详见本目录下的 `mcp-tools-map.md` 中的映射表。文件名格式为 `[序号]-[语义名称].md`。

## 缓存刷新策略

| 数据类别 | 刷新周期 | 说明 |
|---|---|---|
| 基本信息（股票资料、基金资料、债券发行信息等） | 同一自然日 | 基础资料变更频率低 |
| 财务报表数据 | 同一自然日 | 季报/年报驱动，变更频率按报告期 |
| 日频行情数据 | 同一自然日 | 每日收盘后更新 |
| 估值指标（市盈率、市净率等） | 同一自然日 | 随行情变动，但同一天内基本稳定 |
| 新闻/公告 | 同一自然日 | 新闻公告持续更新，但同一次搜索同天结果相近 |
| 宏观经济指标 | 同一自然日 | 通常按月度/季度发布，日变更极少 |
| 高频行情（highfreq_quotes） | **不缓存** | 实时数据，每次都必须重新查询 |
| 股东/持有人数据 | 同一自然日 | 按报告期披露，变更频率低 |

- 如需强制刷新某个数据项，删除对应缓存文件即可
- 跨自然日后，旧缓存自动失效，重新查询后覆盖
