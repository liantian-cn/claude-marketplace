# Tavily Research — 参数参考

AI 驱动的深度研究，收集多个来源、分析并生成综合报告。耗时 30-120 秒。

## MCP 工具

`tavily_research`

## 参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `input` | string | (必填) | 研究内容的全面描述 |
| `model` | string | `auto` | 研究深度：`mini`、`pro`、`auto` |

## 模型选择

| 模型 | 适用场景 | 典型耗时 |
|------|----------|----------|
| `mini` | 单主题、聚焦研究 | ~30s |
| `pro` | 全面多角度分析、对比 | ~60-120s |
| `auto` | 让 API 根据复杂度决定 | 不定 |

**经验法则：**
- "X 是什么？" 或 "介绍一下 Y" → `mini` 通常足够
- "对比 X vs Y vs Z" 或 "分析…市场" → `pro` 以获得全面覆盖
- 不确定时 → `auto`（API 自行选择）

## 使用技巧

- **研究需要时间** — 至少 30 秒，复杂主题可达 2 分钟
- **input 描述越具体越好** — 提供越多上下文，研究质量越高。包含需要关注的角度、需要的信息类型、用途
- **复杂对比或多面主题用 `model=pro`** — 当全面性比速度更重要时
- **快速事实查询用 `tavily_search`** — research 用于深度综合，不是快速查询
